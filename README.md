# ExpressionSolver
Консольный калькулятор, вычисляющий арифметические выражения, состоящих из простейших арифметических операций (+, -, *, /, скобки), и сообщает об ошибках, которые были допущены в выражении.
## Покрытие Unit-тестами - 65%
